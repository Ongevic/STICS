# LORO Repro

LORO reproduction is the truth serum page. This is where we stop talking about
whether a workflow should work and check whether it actually reproduces the
numbers we claim.

## Mission

Track the state of local leave-one-round-out reproduction across STICS, APSIM,
and DSSAT, with specific dates and file evidence.

## Why this page matters

- reproducibility status changes over time

## Operating rule

Every repro note should include:

- date
- exact launcher
- output artifact paths
- current status
- next step

That keeps this page from turning into folklore.
